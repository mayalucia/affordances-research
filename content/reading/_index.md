## What this section is

References for the work, in three classes:

- **Foundational** — Gibson, Warren, Turvey. The affordance theory
  primary literature.
- **Adjacent** — Friston, Tishby, Pitkow, Lengyel. Information-
  theoretic and free-energy work that touches the same questions
  from different angles.
- **Recent** — Schrimpf, Marin-Vargas. Brain-score and primate-hand
  recordings; the contemporary literature we read directly.

Each entry comes with two sentences:

1. **What it gets right** — the contribution, in one sentence.
2. **What we don't take from it** — the position we don't share, or
   the move we don't follow.

The second sentence is the discipline. A reading list that only
endorses is not yet honest.

## Cite-or-strike status

Every entry has been checked against the actual published text, not
against a memory of it. If we have not read it, we do not cite it.
The process is documented in [Method → Coalition](/method/coalition/);
the discipline is documented in [Method → Disciplines](/method/disciplines/).

The check is run before commit using the cite-or-strike power doc
in the parent project (`aburaya/powers/cite-or-strike.md`).
